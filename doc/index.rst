**Hipsta**
==========

.. toctree::

    documentation/index
    tutorials/index
    changes/index
    api/index

Hipsta is an `open-source Python package <project github_>`_ for hippocampal
shape and thickness analysis.


Install
-------

Hipsta is available on `Pypi <project pypi_>`_ and can also be installed from
source.

.. tab-set::

    .. tab-item:: Pypi

        .. code-block:: bash

            pip install hipsta

    .. tab-item:: Source

        .. code-block:: bash

            pip install git+https://github.com/Deep-MI/Hipsta
